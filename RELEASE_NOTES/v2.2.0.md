## Neler Geldi?

Bu sürümde uygulamanın tüm kullanıcı arayüzü ve backend API'ları tamamlandı.

### Yeni Ekranlar

- **Onboarding:** Animasyonlu globe, yüzen dil rozeti, "Hemen Başla" butonu
- **Home:** Token bakiye kartı, mod seçimi (Standart/Hızlı/Pro), son oturumlar listesi
- **LiveTranslation:** Gerçek zamanlı dalgaform animasyonu, transkript, istatistik satırı
- **TokenShop:** 3 paket (Starter 500/50₺, Popular 1200/100₺, Pro 3000/200₺), satın alma akışı
- **Profile:** Avatar, istatistik grid, başarımlar, ayar toggle'ları

### Backend API'lar

| Endpoint | Açıklama |
|----------|----------|
| `GET /api/user/profile` | Kullanıcı profili |
| `PUT /api/user/profile` | Profil güncelle |
| `GET /api/user/tokens` | Token bakiyesi |
| `GET /api/sessions` | Oturum listesi (max 20, yeniden eskiye) |
| `POST /api/sessions` | Yeni oturum kaydet |
| `DELETE /api/sessions/{id}` | Oturum sil |
| `GET /api/sessions/stats` | Toplam istatistik |
| `GET /api/tokens/packages` | Paket listesi |
| `POST /api/tokens/purchase` | Token satın al |
| `POST /api/tokens/deduct` | Token düş |
